# FHIR

Endpoint base url: `https://release.openimis.org/api/api_fhir_r4/`

## Related docs

- OpenIMIS wiki entry: https://openimis.atlassian.net/wiki/spaces/OP/pages/1233649676/openIMIS+FHIR+R4+Overview+Page
- OpenIMIS FHIR documentation website: http://fhir.openimis.org/index.html
- Backend module: https://github.com/openimis/openimis-be-api_fhir_r4_py/

## Login request

HTTP request:

```yaml
endpoint: https://release.openimis.org/api/api_fhir_r4/login
method: POST
headers:
  Content-Type: application/json
body: { "username": "admin", "password": "redacted" }
```

Sample response:

```json
{
  "token": "redacted",
  "exp": 1649841556
}
```

## Request to fetch data (sample: patients)

HTTP request:

```yml
endpoint: https://release.openimis.org/api/api_fhir_r4/Patient
method: GET
headers:
  Content-Type: application/json
  Authorization: Bearer <jwt-token-vom-login-request>
```
